To the end of this project... and beyond!

A few thoughts about developing my first project in HTML and CSS from scratch. Do I feel like a loser or like a ninja? Spoiler: none of them.


And both of them. What I am happy about my project?

It is really interesting to build a website from scratch: the “WTF” given by the blank page turns into “hey, I get this” from a day to another. The awareness I gained is really precious to improve my work, web architecture in primis.

The things I got stuck into

I have always been a content person: I write since I got that my drawing skills are really bad. That means, since forever! For me it is much easier to think “Flex” than “Grid”: next time I need to focus on the layout better to avoid to be stuck into problems, solving them on the run with ehm...not so nice code. This hurts!
